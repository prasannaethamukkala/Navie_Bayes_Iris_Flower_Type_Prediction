# Navie_Bayes_Iris_Flower_Type_Prediction
# 🌸 Iris Flower Prediction Web App

A simple Machine Learning web application built using **Flask** that predicts the type of Iris flower based on input features.

---

## 🚀 Project Overview

This project uses a **Naive Bayes classification model** to predict the species of an Iris flower:

* 🌼 Setosa
* 🌺 Versicolor
* 🌸 Virginica

The user provides input values like:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

and the model predicts the flower type.

## ⚙️ How to Run the Project

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/iris-flower-prediction.git
cd iris-flower-prediction
```

### 2️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Run the Flask app

```
python app.py
```

### 4️⃣ Open in browser

```
http://127.0.0.1:5000/
```
